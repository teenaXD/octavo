# octavo
Me and my teammates build this e-commerce website as our First Year Computer Engineering project for purchasing books online. It allows people to access a large number of eBooks presented in a simple, efficient manner. The main focus of the project is creating a functional website that does not sacrifice in design language. Currently, without a backend the website uses redirects for users to make their purchases.

In order to run this project open the "Home" file on your browser.
